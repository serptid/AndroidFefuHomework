SplitMate — Android-приложение для разделения общего счёта между несколькими людьми.

Пользователь вводит сумму счёта и количество людей, после чего приложение рассчитывает чаевые (10%), итоговую сумму и сумму на одного человека.

---

Экраны приложения

Home  
Стартовый экран с кнопками Start и History.

Место для скриншота Home

Input  
Экран ввода данных:
- Total — сумма счёта
- People — количество людей

Кнопка Calculate активна только при корректном вводе.

Место для скриншота Input

Result/{calcId}  
Экран результата, открывается по маршруту с аргументом calcId.  
Показывает чаевые, итоговую сумму и сумму на человека.  
Доступны кнопки Back, New и Share.

Место для скриншота Result

History  
Экран истории с последними 5 расчётами.  
Нажатие открывает соответствующий результат.

Место для скриншота History

---

Пример работы

Total = 1000  
People = 4

Результат:
- чаевые = 100
- итоговая сумма = 1100
- на одного человека = 275
